# 2020铝型材床头柜加工与采购清单

## 设计基准

- 外形尺寸：400mm 宽 × 400mm 深 × 420mm 高。
- 主框架：2020 铝型材，三层框架，四根立柱。
- 上层：木抽屉 + 左右/后侧 4mm 长虹玻璃或亚克力；侧板嵌入型材框内，板边由槽/U型胶条/压片遮住，只露出长虹纹理面。抽屉滑轨不固定在亚克力上，需固定在隐藏金属承载条上。
- 下层：18mm 置物板嵌入底层 2020 内框，板面与铝型材上表面齐平；外侧四面 8mm 不锈钢光轴护栏。
- 模型文件：`床头柜建模.scad`，已补入加工尺寸、孔位、滑轨、抽屉板件和主要五金位置。
- 单独导出零件时可修改 `render_target`，支持 `assembly`、`frame`、`top_plate`、`glass_panels`、`shelf`、`guard_rails`、`drawer`、`fixed_slides`、`drawer_install`、`drawer_closed`、`drawer_open`、`feet`。
- 型材接口按隐藏盲连/内置连接件处理，外侧不要布置外露黑色螺丝。

## 加工前复核点

1. 顶板孔中心按模型是距外边约10mm，适配2020立柱中心孔；下单前让金属板加工方和型材方同时确认。当前模型按 M5 沉头孔：通孔约5.5mm，沉头直径约10.5mm，沉头深约2.5mm，建议使用3mm金属板。
2. 顶板沉头螺丝建议用银色不锈钢件，拧紧后与不锈钢面板基本齐平；如果改 M6，必须同步改通孔、沉头直径、顶板厚度和立柱攻牙规格。型材顶端攻牙规格、底部调节脚螺纹规格必须由同一家型材卖家确认，不能默认M5/M6通用。
3. 顶板外轮廓不要按大 R4 或大倒角处理；建议按实际2020铝型材外角做 R1.5-R2，四周只做 C0.3-C0.5 轻微破边。这样顶板外轮廓能覆盖立柱外角，不会从上方看到下方铝型材角。
4. 如果卖家不支持2020隐藏盲连/内置连接件，改用角码会破坏参考图这种无外露螺丝效果，螺丝和滑块数量也要重新计算。
5. 抽屉滑轨实际高度、厚度、孔位差异较大；当前模型按350mm全拉出滑轨计算，关闭时抽屉前脸外表面与前侧铝型材外表面平齐，抽屉后方保留约12mm安全间隙。买到具体滑轨后，应把 `slide_h`、`slide_thk`、`slide_len` 回填到 `床头柜建模.scad`，并保持低位贴近抽屉盒底边。滑轨必须可拆内轨，否则侧板装好后很难完成装配和维护。
6. 滑轨不要安装在亚克力/长虹板上。推荐装配顺序是：先装2020框架并预放T型滑块；再固定左右隐藏承载条；再从抽屉口方向把滑轨外固定轨锁到承载条；抽屉盒在柜体外组装并锁活动内轨；最后从正面推入合轨并调缝。侧面长虹板/亚克力建议在抽屉滑轨试装顺畅后再最终锁紧。
7. 当前模型按 3mm 承载条 + 12mm 滑轨 + 每侧2mm装配/调节余量计算，抽屉盒外宽为326mm。若实际滑轨厚度或承载条厚度变化，抽屉盒外宽应按 `360 - 2 × (承载条厚度 + 滑轨厚度 + 每侧装配余量)` 重新计算；每侧装配余量不建议小于1.5mm。
8. 如果坚持使用原清单的300mm滑轨，抽屉盒深度应改回300mm，完全关闭后抽屉后方会留约60mm空隙；这不是装不上的问题，而是储物深度利用不足。
9. 抽屉前脸不要和抽屉盒一次性死锁；建议用4点可调固定件或长孔+大垫片，整柜试装后再调平、调缝、锁紧。
10. 长虹玻璃建议倒角或磨边；亚克力更好加工，但耐刮性弱于玻璃。侧面板不要外凸安装，板边应藏进型材框或U型胶条。
11. 底层置物板不要放在型材上方；应下沉嵌入底层2020内框，顶面和底层横梁上表面齐平。底板下方至少四处使用槽内式承托片承重，压片/卡条只负责防上浮和定位。
12. 如果底板承重要放书、设备或较重物品，建议把底板槽内承托片从4个增加到6-8个，前后边各2-3个，左右边按需要补强。
13. 光轴固定件建议采购“2020槽内式8mm圆杆夹/通用光轴连接件”，通常由圆杆夹体 + M5螺丝 + T型滑块组成，能直接嵌入型材槽。
14. 所有板件建议保留0.5-1mm加工误差余量，先试装框架，再锁紧面板和抽屉。

## Windows/MayCAD 导出说明

- OpenSCAD 当前可直接导出 `STL`、`3MF`、`AMF`、`OFF`、`CSG` 等格式；本项目优先导出 `STL` 和 `3MF` 给 Windows 端确认。
- 本机没有 FreeCAD/CadQuery，因此不直接输出 STEP。若你的 Windows MayCAD 只支持 STEP/IGES，需要在 Windows 端用 FreeCAD、Fusion、SolidWorks 或 MayCAD 支持的转换工具从 STL/3MF 另转，或者提供 MayCAD 的明确导入格式后再重新导出。
- `STL` 是网格模型，适合确认空间、外观、干涉和装配关系；真正下料仍以本清单尺寸和 `床头柜建模.scad` 参数为准。
